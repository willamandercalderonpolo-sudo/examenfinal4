# Sistema de Reservas para Cafetería Universitaria

## Descripción
Sistema en C# para gestionar reservas de combos en una cafetería universitaria con dos turnos (mañana y tarde).

## Funcionalidades
- Registrar reservas de combos
- Cancelar reservas
- Listar todas las reservas
- Ver reportes de ingresos
- Buscar reservas por nombre

## Requisitos
- .NET Framework 4.7.2 o superior
- Visual Studio 2019+

## Ejecución
1. Abrir el proyecto en Visual Studio
2. Presionar F5 o hacer clic en Iniciar
3. Seleccionar una opción del menú
